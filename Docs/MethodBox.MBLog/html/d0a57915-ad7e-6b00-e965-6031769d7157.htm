      MethodBox.MBLog.Logging 命名空间

      类

      Logger 表示MBLog中的日志系统执行类。
      LogTracker 用于进行日志跟踪的跟踪器类。

      委托

      LoggerDangerousLogHandler 表示当产生警告和错误等级的日志时引发的事件所对应的委托签名。
      LoggerDataTrackHandler 表示当产生捕获到指定类型的日志时引发的事件所对应的委托签名。
      LoggerLogStringHandler 表示一个可用于自定义生成日志字符串方法的委托。
